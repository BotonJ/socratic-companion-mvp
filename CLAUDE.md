# 学习伴侣系统

## 项目简介

一个基于Prompt驱动的轻量级AI学习伴侣系统，融合苏格拉底教学法和虚拟角色扮演，让学习变得高效有趣。

## 项目背景

本系统基于以下三个实战方案设计：
1. **吴乐旻的"苏格拉底·七"系统**：理论框架创始人，苏格拉底教学法+二次元角色
2. **洛星尘的《与行星相会》系统**：DeepSeek 1M上下文实战，天体力学学习
3. **胡魏的经济学课堂实录**：Cursor+Claude Code实战，命令化启动+自动更新

## 核心设计

### 教学模式
- **苏格拉底教学法**：问题驱动，学生自己推理
- **多角色互动**：3人团队，分工明确（主讲/排雷/降维）
- **情感连接**：基于"温情脉脉的朋友之谊"

### 技术架构
- **Prompt驱动**：无需复杂编码，纯MD文件配置
- **文档化管理**：进度、人设、日记全部文档化
- **命令化启动**：通过命令快速启动学习会话

## 项目结构

```
projectc/
├── templates/           # Prompt模板库
│   └── system.md      # 系统核心（最高指令）
│
├── personas/           # 角色人设库
│   └── sanqyue-seven.md # 三月七人设（示例）
│
├── session/            # 会话状态
│   ├── progress.md    # 学习进度
│   └── diary.md       # 学习日记
│
├── agents/             # Agent定义（预留）
├── skills/             # 技能定义（预留）
└── commands/           # 斜杠命令（预留）
```

## 使用方法

### Phase 1（当前版本）

1. 手动读取 `templates/system.md`
2. 手动读取 `personas/sanqyue-seven.md`
3. 手动读取 `session/progress.md`
4. 开始教学会话

### Phase 2（规划中）

- 添加命令：`/teach-me`、`/read-with`
- 添加角色人设编辑器
- 添加课后自动更新机制
