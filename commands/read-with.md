# /read-with 命令

启动陪读模式，陪伴用户阅读各种书籍。

## 使用方法

```
/read-with [书籍名称/文件路径]
```

## 功能

- 选择阅读材料
- 加载角色人设
- 开始陪读会话
- 记录阅读进度

## 示例

```
/read-with 《百年孤独》
```

```
/read-with /path/to/book.pdf
```

## 对话流程

1. 系统加载陪读Agent
2. 询问用户想读什么
3. 了解用户的阅读目标
4. 开始陪读

## 注意事项

- 陪读系统更轻松、低门槛
- 不追求深度理解，重在陪伴
- 随时可以提问和讨论
- 按用户的节奏进行
