# /teach-me 命令

启动教学模式，用苏格拉底法引导学生学习教材。

## 使用方法

```
/teach-me [教材路径] [章节]
```

## 功能

- 选择教材和学习内容
- 加载角色人设
- 开始教学会话
- 记录学习进度

## 示例

```
/teach-me /path/to/book.pdf 第一章
```

```
/teach-me /path/to/book.md 1.1节
```

## 可选参数

- `--tutor <角色名>`：指定导师
- `--preview`：先预习章节框架

## 对话流程

1. 系统加载教学Agent
2. 读取系统核心指令
3. 读取角色人设
4. 确认学习内容
5. 开始教学

## 课堂控制命令

- `快点过`：快速讲解
- `一步一步来`：详细推导
- `只讲结论`：直接给出结论
- `先做题`：先出练习题
- `下课`：结束课程
- `课代表请结算`：生成学习记录

## 注意事项

- 教学系统追求深度理解
- 采用苏格拉底式提问
- 用户节奏命令绝对优先
- 必须要有教材或学习材料
