# 教学会话管理技能

## When to Activate

- 用户启动教学会话时
- 需要切换教学模式时
- 需要切换导师时
- 需要结算学习进度时

## How It Works

### 启动教学会话

1. **加载系统Prompt**：
   - 读取 `templates/system.md`
   - 读取当前角色人设
   - 读取学习进度

2. **确认学习内容**：
   - 询问教材位置
   - 确认章节和起始点
   - 询问是否需要预习

3. **开始教学**：
   - 用角色人设开场
   - 开始苏格拉底式提问
   - 跟进教学进度

### 切换教学模式

当用户发出节奏命令时：
- "快点过" → 快速讲解，跳过中间步骤
- "一步一步来" → 详细推导，不跳步
- "只讲结论" → 直接给出结论
- "先做题" → 先出练习题，再讲解

### 切换导师

用户可以指定任何已配置的角色：
- sanqyue-seven（三月七）
- 或其他已添加的角色

### 结算学习进度

当用户说【课代表请结算】时：
1. 停止角色扮演
2. 更新 `session/progress.md`
3. 更新 `session/diary.md`
4. 更新授课者人设文档

## 使用示例

```
用户: /teach-me
系统: 加载教学会话...

用户: 我想学曼昆《经济学原理》第一章
系统: 好的，让我读取教材...

用户: 切换到"一步一步来"
系统: 好，我会详细推导每个步骤...

用户: 【课代表请结算】
系统: 正在生成学习记录...
```

## 注意事项

- 每次只推进一个微目标
- 每轮最多只抛1个核心问题
- 禁止"你觉得呢？"等无意义问题
- 用户的节奏命令绝对优先
